Capas Del Modelo Osi

Capa Física
• Función: Define las características físicas de la conexión, como cables, tarjetas de red, conexiones de fibra óptica, voltajes, frecuencias, etc.
• Ejemplos: Cables de cobre (Ethernet), señales de radio (Wi-Fi), conectores, modems.
Capa de Enlace de Datos
• Función: Garantiza la transmisión fiable de datos entre dos dispositivos en la misma red local. Se encarga del control de acceso al medio (como la asignación de direcciones MAC), y de la corrección de errores.
• Ejemplos: Ethernet, Wi-Fi, PPP (Point-to-Point Protocol), ARP (Address Resolution Protocol).
Capa de Red
• Función: Se encarga del enrutamiento de los datos entre redes distintas. Define las direcciones lógicas (como las direcciones IP) y controla el direccionamiento y la fragmentación de los paquetes.
• Ejemplos: IPv4, IPv6, enrutadores, ICMP (Internet Control Message Protocol).
Capa de Transporte
• Función: Garantiza que los datos se entreguen de manera correcta y en el orden adecuado. Se encarga del control de flujo y la detección de errores en los datos que se transmiten entre los sistemas finales.
• Ejemplos: TCP (Transmission Control Protocol), UDP (User Datagram Protocol).
Capa de Sesión
• Función: Establece, mantiene y termina sesiones entre aplicaciones. Controla el diálogo (conexión) entre dos sistemas y se asegura de que la comunicación sea continua y no se pierdan los datos durante la transmisión.
• Ejemplos: Protocolos de control de sesión, como el protocolo RPC (Remote Procedure Call).
Capa de Presentación
• Función: Se encarga de la representación de los datos. Traduce, comprime y cifra los datos para que puedan ser entendidos por la aplicación que los recibe.
• Ejemplos: SSL/TLS (para cifrado de datos), JPEG, GIF, ASCII, EDI (Intercambio Electrónico de Datos).
Capa de Aplicación
• Función: Es la capa más cercana al usuario y donde residen las aplicaciones de red. Permite que los usuarios interactúen con la red a través de aplicaciones como correo electrónico, navegación web, transferencias de archivos, etc.
• Ejemplos: HTTP (para navegar por la web), FTP (Transferencia de Archivos), SMTP (Correo Electrónico), DNS (Sistema de Nombres de Dominio).
